================================================================================

1. Open and read: GET_STARTED.md
   → This has EVERYTHING you need, step-by-step

2. Install dependencies:
   pip install -r requirements.txt

3. Create .env file with your:
   - Kalshi credentials
   - Gmail credentials
   (See GET_STARTED.md for template)

4. Test:
   python3 main.py --mode once

5. Run continuously:
   python3 main.py --mode scheduled

WHAT THIS DOES
================================================================================

1. Scans Polymarket & Kalshi every 5 minutes
2. Finds matching markets (fuzzy matching)
3. Detects arbitrage opportunities (guaranteed profit both outcomes)
4. Emails you when profit >= $100
5. Tells you exactly how much to bet on each platform
